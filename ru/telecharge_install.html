1. Загрузка и установка

blunderDB — это единственный исполняемый файл, не требующий установки.

Последняя версия blunderDB доступна по лицензии MIT:

  • для Windows: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-windows-0.28.0.exe

  • для Linux: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-linux-0.28.0

  • для Mac: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-macos-0.28.0.zip

Для работы в режиме офлайн вы также можете скачать данную документацию в формате PDF: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-0.28.0-fr.pdf

Примечание

blunderDB использует Webview2 для отображения графического интерфейса. Вполне вероятно, что Webview2 уже установлен в вашей операционной системе. Если нет, при первом запуске blunderDB предложит скачать и установить его. Никаких действий со стороны пользователя не потребуется.

1.1. Установка в Linux

Для Linux предлагается несколько форматов. Приведённые ниже пакеты и архивы делают blunderDB исполняемым автоматически: в отличие от необработанного двоичного файла, загруженного через браузер, они избавляют от необходимости выполнять chmod +x при каждой загрузке или обновлении. Они также создают пункт в меню приложений.

1.1.1. Нативные пакеты (.deb / .rpm)

Рекомендуемый способ в Debian, Ubuntu и Linux Mint (.deb), а также в Fedora и openSUSE (.rpm). Менеджер пакетов автоматически устанавливает подходящую зависимость webkit2gtk. Замените x.y.z на загруженную версию:

sudo apt install ./blunderdb_x.y.z_amd64.deb     # Debian / Ubuntu / Mint
sudo dnf install ./blunderdb-x.y.z.x86_64.rpm    # Fedora / openSUSE

1.1.2. Arch Linux (AUR)

Пакет blunderdb-bin доступен в AUR и автоматически обновляется помощниками AUR:

yay -S blunderdb-bin      # ou : paru -S blunderdb-bin

1.1.3. Архив .tar.gz

Для других дистрибутивов. При извлечении архива сохраняется бит исполнения, поэтому chmod не требуется:

tar xzf blunderDB-linux-webkit2gtk-4.1-x.y.z.tar.gz
cd blunderDB-linux-webkit2gtk-4.1-x.y.z
./blunderdb

1.1.4. Необработанный двоичный файл (продвинутый способ)

Необработанный двоичный файл https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-linux-0.28.0 остаётся доступным. Поскольку браузер удаляет бит исполнения при загрузке, его необходимо восстановить перед первым запуском:

chmod +x ./blunderDB-linux-x.y.z

Примечание

Публикуются два варианта Linux в зависимости от версии библиотеки webkit2gtk. Если вы получаете ошибку libwebkit2gtk-4.0.so.37: cannot open shared object file, ваш дистрибутив использует webkit2gtk-4.1: используйте пакет .deb/.rpm, пакет AUR или загрузите специальную сборку https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-linux-webkit2gtk-4.1-0.28.0. Нативные пакеты выбирают правильную зависимость автоматически.

1.2. Предупреждения для Windows и Mac

Предупреждение

В Windows система может проявлять нежелание запускать blunderDB. Смотрите Раздел 10 для понимания причин и обхода возможных блокировок.

Предупреждение

На Mac система может проявлять нежелание запускать blunderDB. Смотрите Раздел 11 для понимания причин и обхода возможных блокировок.
