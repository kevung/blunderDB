1. Загрузка и установка

blunderDB — это единственный исполняемый файл, не требующий установки.

Последняя версия blunderDB доступна по лицензии MIT:

  • для Windows: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-windows-0.27.1.exe

  • для Linux: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-linux-0.27.1

  • для Mac: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-macos-0.27.1.zip

Для работы в режиме офлайн вы также можете скачать данную документацию в формате PDF: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-0.27.1-fr.pdf

Примечание

blunderDB использует Webview2 для отображения графического интерфейса. Вполне вероятно, что Webview2 уже установлен в вашей операционной системе. Если нет, при первом запуске blunderDB предложит скачать и установить его. Никаких действий со стороны пользователя не потребуется.

Примечание

В Linux, если blunderDB не является исполняемым после загрузки, выполните в терминале команду chmod +x ./blunderDB-linux-x.y.z, где x, y, z соответствует загруженной версии.

Примечание

В Linux, если вы получаете ошибку libwebkit2gtk-4.0.so.37: cannot open shared object file, ваш дистрибутив использует webkit2gtk-4.1 вместо webkit2gtk-4.0. Скачайте специальную сборку: https://github.com/kevung/blunderDB/releases/latest/download/blunderDB-linux-webkit2gtk-4.1-0.27.1

Предупреждение

В Windows система может проявлять нежелание запускать blunderDB. Смотрите Раздел 10 для понимания причин и обхода возможных блокировок.

Предупреждение

На Mac система может проявлять нежелание запускать blunderDB. Смотрите Раздел 11 для понимания причин и обхода возможных блокировок.
